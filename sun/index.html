给博客添加个太阳升起效果

2011-06-18

我博客右边那个太阳升起的特效是我去年某个阳光灿烂的午后从某个博客偷偷拿过来的， 我也忘记具体是哪个博客了，今天有人想要这个效果，就发出来方法吧。

直接把代码放一起了，添加到 header.php 或者 footer.php 中就可以了， 已经加载过 jquery 的，就删除第一行的吧。

<script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jquery/1.4.4/jquery.min.js"></script>
<script type='text/javascript'>
function sunset() {
    $("#sun").css("top", 500).css("right", 0).animate({
        top: 10,
        right: 80
    },
    15000)
}
$(function() {
    sunset()
})
</script>
<style type="text/css">
.sun {background: url(http://ss.last.cc/sun.png) no-repeat;height: 100px;position: absolute;right: 80px;top: 10px;width: 100px;}
</script>
<div id="sun" class="sun" style="top: 15.9004px; right: 79.0367px; "></div>
