Pinterest Clone Scripts

2013-05-03

网上找了一遍像样的 Pinterest Clone Script, 发现居然没有, 有个pinry, 太简陋了, 剩下的 几乎都是收费的.

看了这些收费的,很多还是基于 wordspress 改得, 性能不会太好, 很奇怪的是 Pinterest 这个应用并不负载, 为什么没有开源的实现呢 ? 难道是 Pinterest 太好了, 都不忍 clone ?

这里有些 pinterest clone script 的列表, 喜欢的可以参考: http://www.cloneidea.com/pinterest/all/all

另外,目测还是 ruby 社区活力比较大,我同样的问题发到 其他社区都没人理, 好要ruby社区有人回答 :)
